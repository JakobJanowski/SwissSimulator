# Swiss Simulator

An application which allows you to generate brackets for a Swiss Styled Tournament.

Explanation of [Swiss styled Tournaments](https://en.wikipedia.org/wiki/Swiss-system_tournament)

## Instalation Instructions

Download and extract the zip file from the realese section and run the SwissSim.exe.

Project was made in Godot 4.3 mono. Please use Godot 4.3 or later to open source files.
